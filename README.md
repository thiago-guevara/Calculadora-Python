# Calculadora-Python
Calculadora básica desarrollada en Python utilizando estructuras condicionales y repetitivas.
## Funcionalidades

- Suma de dos números.
- Resta de dos números.
- Multiplicación de dos números.
- División de dos números.
- Validación para evitar división entre cero.
- Menú interactivo.
- Opción para salir del programa.

## Herramientas utilizadas

- Python 3
- Visual Studio Code
- GitHub

## Lógica del programa

El programa muestra un menú con diferentes operaciones matemáticas. El usuario selecciona una opción e ingresa dos números. Dependiendo de la opción elegida, se realiza el cálculo correspondiente y se muestra el resultado en pantalla.

El programa utiliza estructuras condicionales (`if`, `elif`, `else`) para seleccionar la operación matemática y una estructura repetitiva (`while`) para mantener activo el menú hasta que el usuario decida salir.
